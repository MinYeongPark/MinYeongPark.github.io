# 컴퓨터시스템관리 1주차 실습일지

컴퓨터공학과 20191062 박민영



1. 실습과제 결과

https://youtu.be/qVMcMASibkQ



2. 새로 배운 내용

   - 가상머신이란, 한 대의 컴퓨터에서 여러 대의 컴퓨터를 운영하는 것처럼 만드는 방법을 말한다.

   - VM ware 가상머신 소프트웨어와 리눅스의 기초 사용법에 대해 배웠다.

   - 리눅스 운영체제를 사용하는 Server, Server(b), Client의 총 3대의 컴퓨터를 만들고, 설치하였다.

     

3. 문제가 발생하거나 고민한 내용 + 해결 과정

   Server(b)를 만들었을 때 처음에는 RAM 용량을 512MB로 설정하고 실습을 진행했었으나, 다음 상황으로 진행되지 않고 동작이 멈추게 되었다. <컴퓨터시스템관리> 수업의 Slack qna에 다른 학우님께서 Server(b)의 RAM 용량을 1GB로 변경하면 정상적으로 잘 작동한다는 글을 올려주셨고, 한번 이 방법을 이용해보았는데 정상적으로 작동하게 되었다.

   RAM 메모리 크기 설정하는 곳에서 최소 1GB는 추천한다고 쓰여있는데, 이러한 이유때문에 오류가 발생했을 것이라고 추측한다.

4. 참고할 만한 내용

   Server와 Server(b)는 ubuntu로 설치했고, Client는 kubuntu로 설치했다. ubuntu와 kubuntu가 서로 어떤 점에서 차이점이 있는지 조사해보았다.

   먼저 우분투는, 데비안 리눅스를

   [^데비안 리눅스]: 유명한 리눅스 배포판 중 하나

    기초로 만든 리눅스 배포판으로, 다른 배포판에 비해 설치가 편리하고 유지보수 절차가 편하고, 사용자에게 친화적인 인터페이스를 제공함으로써 가장 넓은 사용층을 가진 배포판 중 하나이다. 개인용, 데스크탑 환경에 최적화되어있고, 사용자 편의를 중점으로 개발되고 있다.

   우분투의 첫 번째 버전은 GNOME 데스크탑 환경을 기반으로 만들어졌으나, 이후 GNOME 대신 KDE를 GUI로 사용하는 쿠분투가 만들어졌다. 이로써 마이크로소프트의 윈도우와 비슷한 환경이 구축되었다. 

   참고 : [기초지식] 우분투란 무엇인가? 1 (https://cafe.naver.com/lifevside/26)

5. 회고 (+, -, !)

   좋았던 점 (+) : 정말 낯설었던 가상머신과 리눅스를 직접 설치해보고 실습해보면서 이해할 수 있게 되었다. 아직 설치와 부팅, 설정, 종료만 해보았지만, 직접 구동해보면서 이 <컴퓨터시스템관리> 수업의 전체적인 뼈대가 머릿속에 잡히게 되었다. 앞으로 이 가상 컴퓨터 3대로 진행하게 될 실습들이 기대된다.

   아쉬운 점 (-) : 데스크탑과 노트북 두 컴퓨터에 동시에 설치를 진행했는데 두 컴퓨터의 설치 속도가 달라서 더 오래 걸린 점이 조금 아쉬웠다. 그래도 이렇게 오랜 시간 힘들게 설치한 만큼 앞으로의 실습에 더 열심히 임할 수 있을 것 같다.

   새로 알게 된 점(!) : 리눅스는 그 자체로 한 종류의 운영체제인 줄 알았는데 그 안에서도 ubuntu, kubuntu 등등 종류가 매우 다양하다는 것을 알게 되었다. 가상머신이 무엇인지 잘 몰랐었는데, 한 대의 컴퓨터에서 여러 대의 컴퓨터를 운영하는 것처럼 만드는 방법이라는 것을 알게 되었고, 실습을 진행하면서 제대로 이해하게 되었다.
